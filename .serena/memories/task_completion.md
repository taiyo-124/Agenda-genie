Before committing any code, ensure that the following commands run successfully without any errors:

1.  **Format the code**:
    ```bash
    ruff format .
    ```
2.  **Check for linting issues**:
    ```bash
    ruff check .
    ```
3.  **Check for type errors**:
    ```bash
    mypy .
    ```
4.  **Run all tests (once implemented)**:
    ```bash
    pytest
    ```
